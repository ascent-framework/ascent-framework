# Archive

This directory contains archived material from earlier stages of the project.

## Current Contents

- `operator-coordinate-compression/`: documents preserved from the predecessor repository
- `ascent-framework-notes/`: preserved raw note exports from the ASCENT draft workspace

## Purpose

- Keep historically important documents without mixing them into the active framework docs
- Preserve the research lineage from the earlier compression-centered framing to ASCENT
- Provide a stable place for prior-art notes, early theory drafts, and patent-oriented materials

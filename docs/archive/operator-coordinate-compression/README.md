# operator-coordinate-compression Archive

This directory preserves selected documents from the earlier
`operator-coordinate-compression` repository.

## What This Is

These files capture the project phase that framed LLM compression as a
coordinate-system selection problem. They are kept as historical reference
material, not as the current canonical statement of the ASCENT framework.

## Why It Exists

- Preserve the conceptual lineage from the earlier repository to ASCENT
- Keep prior-art and claim-development notes available for later reuse
- Retain early theory documents that still inform the current framework

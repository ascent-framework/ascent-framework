# ASCENT Framework Notes Archive

This directory preserves the ASCENT note trail in a single readable archive
document.

## What This Is

This material comes from the internal drafting stage of the ASCENT framework.
It is preserved for traceability and intellectual history, not as a canonical
framework statement.

## Why It Exists

- preserve the original reasoning trail behind the current framework documents,
- keep a readable Korean archive note for later cross-checking,
- separate historical drafting artifacts from the active `docs/` tree.
